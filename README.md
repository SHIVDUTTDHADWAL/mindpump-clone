# Mind Pump Website Clone

This project is a front-end clone of the Mind Pump website created as part of a coding assignment.  
The goal was to recreate the layout, styling, and main interactions of the website using core web technologies.

## 🚀 Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome
- Google Fonts

## 📱 Features

- Fully responsive layout
- Navigation bar with dropdown menu
- Mobile hamburger menu
- Hero section with call-to-action buttons
- Press/testimonial slider
- FAQ accordion section
- Resources section with download cards
- Embedded YouTube videos
- Blog cards layout
- Newsletter subscription section
- Footer with social media links
